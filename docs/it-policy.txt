1. PASSWORD & ACCOUNT MANAGEMENT
================================================================================

1.1 PASSWORD REQUIREMENTS
-------------------------
- Minimum Length: 12 characters
- Must Include: At least one uppercase letter, one lowercase letter, one number, and one special character (!@#$%^&*)
- Cannot Include: Your name, username, or common dictionary words
- History: Cannot reuse any of your last 10 passwords
- Expiration: Passwords expire every 90 days
- Lockout: Account locks after 5 failed login attempts

1.2 PASSWORD RESET PROCEDURES
-----------------------------
Self-Service Reset:
- Visit https://password.example.com
- Click "Forgot Password"
- Verify identity via backup email or phone
- Create new password meeting requirements
- Password takes effect immediately

Helpdesk-Assisted Reset:
- Call IT Helpdesk: 555-0199
- Provide employee ID and answer security questions
- Temporary password issued (expires in 24 hours)
- Must change password on first login

1.3 MULTI-FACTOR AUTHENTICATION (MFA)
-------------------------------------
- Required for: All employees accessing corporate systems
- Primary Method: Duo Mobile app push notifications
- Backup Methods: SMS codes, hardware tokens (YubiKey)
- Setup: Visit https://mfa.example.com or contact IT
- Lost Device: Call Helpdesk immediately to disable old device

1.4 SINGLE SIGN-ON (SSO)
------------------------
- All corporate applications use SSO via Okta
- Login URL: https://acme.okta.com
- Benefits: One password for all apps
- Supported Apps: Slack, Zoom, Salesforce, Workday, Jira, Confluence, GitHub

1.5 ACCOUNT LOCKOUT
-------------------
If your account is locked:
1. Wait 30 minutes for automatic unlock, OR
2. Visit the IT Kiosk on the 2nd floor of Building A
3. Call Helpdesk at 555-0199
4. Use the self-service unlock at https://unlock.example.com

2. VPN & REMOTE ACCESS
================================================================================

2.1 VPN CLIENT INFORMATION
--------------------------
- Software: Cisco AnyConnect Secure Mobility Client
- Download: https://vpn.example.com (click "Download Client")
- Version Required: 4.10 or higher
- Supported OS: Windows 10/11, macOS 12+, Linux Ubuntu 20.04+

2.2 VPN SERVER ADDRESSES
------------------------
- United States: vpn.example.com
- Europe (GDPR compliant): vpn-eu.example.com
- Asia Pacific: vpn-apac.example.com
- Development/Test: vpn-dev.example.com

2.3 CONNECTING TO VPN
---------------------
Step-by-step instructions:
1. Open Cisco AnyConnect
2. Enter server address: vpn.example.com
3. Click "Connect"
4. Enter your AD username (firstname.lastname)
5. Enter your AD password
6. Approve the Duo push notification on your phone
7. Wait for "Connected" status

2.4 VPN TROUBLESHOOTING
-----------------------
Connection Fails:
- Verify internet connectivity first
- Ensure you're not on a network blocking ports 443, 500, 4500
- Try a different network (coffee shop, phone hotspot)
- Restart the AnyConnect client
- Reboot your computer

Slow Performance:
- Connect to the geographically closest server
- Close bandwidth-heavy applications
- Check if VPN is required (some apps work without VPN)

Split Tunnel:
- VPN only routes corporate traffic by default
- Personal browsing goes through your local connection
- For full tunnel, select "Full Tunnel" profile

2.5 REMOTE DESKTOP ACCESS
-------------------------
- Server: rdp.example.com
- Port: 3389 (default, should be open over VPN)
- Client: Microsoft Remote Desktop (all platforms)
- Access: Request via IT portal, manager approval required

5. NETWORK & WI-FI CONNECTIVITY
================================================================================

5.1 CORPORATE WI-FI NETWORKS
----------------------------
CorpNet-Secure (Employees):
- SSID: CorpNet-Secure
- Authentication: 802.1X with AD credentials
- Auto-connects on corporate devices
- Full network access

CorpNet-IoT (Internet of Things):
- SSID: CorpNet-IoT
- For printers, meeting room displays, etc.
- Limited network access

Guest-Wifi (Visitors):
- SSID: Guest-Wifi
- Password: Changes monthly, ask reception
- Current Password: Posted in lobby
- Limited to internet access only

5.2 CONNECTING TO WI-FI
-----------------------
Windows:
1. Click Wi-Fi icon in taskbar
2. Select "CorpNet-Secure"
3. Enter AD credentials when prompted
4. Check "Connect automatically"

macOS:
1. Click Wi-Fi icon in menu bar
2. Select "CorpNet-Secure"
3. Enter AD credentials
4. Check "Remember this network"

5.3 WI-FI TROUBLESHOOTING
-------------------------
Cannot Connect:
- Verify password is correct
- Forget network and reconnect
- Restart Wi-Fi on device
- Move closer to access point

Slow Connection:
- Check number of devices connected
- Move to less crowded area
- Use Ethernet if available (building lobbies)

Dead Zones:
- Report to IT with exact room number
- Include signal strength reading
- Temporary fix: Use mobile hotspot

5.4 ETHERNET ACCESS
-------------------
- Available in all desks and meeting rooms
- Automatic IP via DHCP
- Faster and more reliable than Wi-Fi
- Use provided Ethernet-to-USB adapter if needed

================================================================================

7.1 ACCEPTABLE USE POLICY
-------------------------
Corporate devices are for business use. Limited personal use permitted if:
- Does not interfere with work
- Does not consume excessive bandwidth
- Does not violate any laws or policies
- Does not store inappropriate content

7.2 DATA CLASSIFICATION
-----------------------
Public:
- Marketing materials, press releases
- No restrictions on sharing

Internal:
- General business documents
- Share only with employees

Confidential:
- Customer data, financial records
- Encrypt when sharing externally
- Manager approval for external sharing

Restricted:
- PII, healthcare data, trade secrets
- Need-to-know access only
- Audit logging enabled

7.3 CLEAN DESK POLICY
---------------------
End of Each Day:
- Lock computer (Windows+L or Ctrl+Cmd+Q)
- Store sensitive documents in locked drawer
- Clear whiteboards of sensitive info
- Shred unnecessary printed documents

7.4 DEVICE SECURITY
-------------------
Required Settings:
- Full disk encryption enabled
- Screen lock after 5 minutes
- Antivirus running (CrowdStrike)
- Firewall enabled

Lost/Stolen Device:
1. Report IMMEDIATELY to IT: 555-0199
2. Device will be remote wiped
3. File police report if stolen
4. Complete incident report form

7.5 USB & EXTERNAL DRIVES
-------------------------
- USB drives must be company-provided and encrypted
- Personal USB drives NOT allowed
- Exception process for specific needs

14. FREQUENTLY ASKED QUESTIONS
================================================================================

Q: How do I connect to Wi-Fi on my first day?
A: Connect to "CorpNet-Secure" and enter your AD credentials (firstname.lastname and password from welcome email).

Q: What's my username format?
A: firstname.lastname (e.g., john.smith)

Q: Can I use my personal laptop for work?
A: Only for accessing email and Slack via browser or enrolled MDM. No company data storage.

Q: How long until my ticket is resolved?
A: Depends on priority. Critical: 4 hours, High: 8 hours, Medium: 24 hours, Low: 5 days.

Q: How do I get Adobe Creative Cloud?
A: Submit Software Access Request at https://requests.example.com. License pool limited, may need to wait.

Q: My VPN keeps disconnecting, what do I do?
A: Try connecting to a different server (vpn-eu.example.com). If persistent, check your internet stability and submit a ticket.

Q: Where can I print in color?
A: Color printers are at designated stations on each floor. Check floor map at https://maps.example.com.

Q: How do I share large files (>25MB)?
A: Upload to OneDrive, then share the link via email instead of attaching the file.

Q: Who do I contact for a security concern?
A: Report immediately to security@example.com or call Security Operations at 555-7373.

Q: How do I request a second monitor?
A: Submit equipment request at https://equipment.example.com. Requires manager approval.

Q: What happens if I forget my password while traveling?
A: Use https://password.example.com self-service, or call Helpdesk at 555-0199 (they're available during US business hours).

Q: Can I install games on my work laptop?
A: Personal software installation is prohibited. See Shadow IT policy.

Q: How often should I restart my computer?
A: At least once per week. This helps apply updates and clear memory.
